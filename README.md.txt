# Resume Screening ML System

This project builds a machine learning system that screens resumes and ranks candidates based on job relevance.

## Features

- Resume text preprocessing using NLP
- Skill extraction
- Resume-job similarity scoring
- Candidate ranking
- Missing skill detection
- Visualization of candidate rankings

## Technologies Used

Python  
Pandas  
Scikit-learn  
spaCy  
NLTK  
Matplotlib  

## Project Workflow

1. Load resume dataset
2. Clean resume text using NLP
3. Convert text to TF-IDF vectors
4. Compare resumes with job description
5. Rank candidates based on similarity score
